# Security Policy

K-MVI is an Android library and does not process secrets directly, but security reports are still welcome.

## Reporting a Vulnerability

Please report suspected vulnerabilities privately by opening a GitHub security advisory for this repository, or by contacting the maintainer through the GitHub profile listed in the README.

Include:

- Affected version or commit
- Impact and attack scenario
- Reproduction steps or proof of concept
- Suggested mitigation, if known

Please do not disclose vulnerabilities publicly until there has been reasonable time to assess and fix them.
